 Для запуска приложения, вам будет достаточно наличие установленной среды Java Runtime Environment. 
Для настройки и запуска Spring Boot приложений требуется следующие версти ПО:
-Java 8+
-Apache Maven 3. x
Посмотреть версию Java и заодно убедиться, что она установлена в вашей системе, можно через терминал с помощью команды:
$java –version


Если среда не установлена, то потребуется скачать архив с необходимой вам версией с официального сайта. Далее вам надо перенести его в желаемую директорию и распаковать. Рекомендуется в директорию /opt/java, далее она и будет использоваться. Сделайте это, используя терминал и следующие команды:

$sudo mv ~/путь/до/файла/jre*.tar.gz /opt/java

$sudo tar -xzf /opt/java/jre*.tar.gz

Где "*" - версия Java.

После установки програмного обеспечения можно запустить jar-файл командой java через терминал  с параметром -jar:
java -jar  имя_файла.jar

Для запуска программы по щелчку мышки или как любую другую Linux-программу - по имени файла необходим пакет jarwrapper, который устанавливается через терминал команой:

$sudo apt install jarwrapper

Теперь можно запускать Jawa в Linux по щелчку мыши или просто командой через терминал :

$ ./имя_файла.jar
